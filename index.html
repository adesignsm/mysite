Today is 3D

This project was made using html, css, and JavScript. Majority of the WebGL content was rendered using THREE.js. The idea behind this project was to initialize user interactions with 3D objects on a web page. I made this project so that I can implement the methods in future projects. It is simple but it comes packed with alot of things like mouse over events and dragging events, all while being used on 3D objects.

Movie Database Project

This project was created using AJAX to make calls to the database, Its main functionality is to pull data from the movie API based on the users input in the search field and displays now playing movies that match the word the user has entered or any movies that include the letters that the user has entered. Used HTML and CSS to structure and style the page along with some jquery for animations, and used AJAX to pull data out of the movie API.

A ToDo List

This project has been created using Node.js, body parsing to parse data, and express. This app has some implementation of AJAX to post and delete tasks from the unordered list elements. A user types in a task and then hits the add task button, the task is then added. Upon hover there is a color animation and once it is clicked the task is deleted.

About Me

My name is Akash Mulye, I began programming and studying design in grade 12. I quickly gained interest in front end web developing/designing scene and it became my greatest passion. Being self taught in front end development I had free reign to practice and explore the fundamentals and the more intricate spectors of this field. After 2 years I decided make an alias for my web work known as AdesignsM, while it is a completely different entity from me it is still a huge part of who I am. AdesignsM is Akash designs Mulye the idea of first name is nothing without its last name and vice versa, same thing with design and development. I now look forward to finding a role in the industry where I can bring my expertise and grow from there.
